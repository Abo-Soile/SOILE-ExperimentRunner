The Filter Node

A Filter Node represents a junction in the SOILE Project or experiment, where the next task depends on the results of previous tasks. A Filter is defined as a function that needs to evaluate to "true".

Filters can use the following functionalities:

  • Operators such as "+", "-", "*", "^" (power)
  • The functions mean(val1, val2, ...), median(val1,val2,...), min(val1,val2,...), max(val1,val2,...)
  • An approximation function that checks, whether two values are the same within a given tolerance: approx(A,B,tol), Where abs(A - B) < tol
  • logical operators: | (or), & (and), = (equal), < (smaller), > (larger)
You can build any formaula with these operators. Best use parenthesis to make operation precedence clear.

The values used in filters need to be defined as outputs in other tasks. and referenced as TaskID.outputName.
